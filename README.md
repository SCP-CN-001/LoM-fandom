# LoM Fandom Site

This repository root is already an initialized Astro project.

Do not run create-astro again in this folder. Running it again can create another nested project directory.

## Run Locally

1. Use the project root:

	/home/rowena/Documents/LoM-fandom

2. Use Node 22:

	nvm use

3. Start dev server:

	npm run dev

## Build And Preview

1. Build:

	npm run build

2. Preview production build:

	npm run preview

## Key Folders

- src
